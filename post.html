componentes y funciones

Estas son las piezas que componen a los computadores y son esenciales para el correcto funcionamiento de este. Para cotizar un computador se debe tener en cuenta que estas sean compatibles entre sí.

Procesador: El procesador es el cerebro del PC, este procesa todo lo que ocurre y ejecuta todas las acciones. Cuanto más rápido sea el procesador que tiene una computadora, más rápidamente se ejecutarán las órdenes que se le den a la máquina.

Tarjeta de video: este componente es el encargado de recibir datos del procesador, transformarlos en imágenes para posteriormente enviar estas últimas a tu monitor, en definitiva se encarga de dar la imagen a tu pantalla.

Memoria RAM: la memoria RAM es componente que se encarga de almacenar los datos de forma temporal. Por lo que cuando el dispositivo se apaga, los datos se borran

Placa madre: es el circuito en el que se conectan los componentes del computador y esta define que tipo de procesador y tarjeta gráfica es compatible con la placa madre.

Fuente de poder: Las fuentes de poder son dispositivos primordiales para el funcionamiento de cualquier equipo, ya que son las encargadas de suministrar la electricidad necesaria para el funcionamiento del computador.

Disco duro: Un disco duro es un dispositivo de almacenamiento necesario para conservar tus archivos y datos en el largo plazo. Siempre que guardas un archivo en la computadora, lo guardas en el disco duro.

SSD: este cumple la misma función del disco duro, pero a diferencia de este, el SSD tiene un sistema de funcionamiento diferente, lo cual lo hace más rápido, más liviano, menos ruidoso y de mayor durabilidad y fiabilidad.
